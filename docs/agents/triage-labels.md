# 트리아지 라벨

스킬은 다섯 개의 정규 트리아지 역할로 말한다. 이 파일은 그 역할들을 이 저장소 이슈 트래커에서 실제로 쓰는 라벨 문자열에 매핑한다.

| mattpocock/skills의 라벨 | 우리 트래커의 라벨 | 의미                          |
| ------------------------ | ------------------ | ----------------------------- |
| `needs-triage`           | `needs-triage`     | 메인테이너가 이 이슈를 평가해야 함 |
| `needs-info`             | `needs-info`       | 리포터의 추가 정보를 기다리는 중 |
| `ready-for-agent`        | `ready-for-agent`  | 완전히 명세됨, AFK 에이전트 준비 완료 |
| `ready-for-human`        | `ready-for-human`  | 사람의 구현이 필요함            |
| `wontfix`                | `wontfix`          | 처리하지 않음                  |

스킬이 역할을 언급하면(예: "AFK-ready 트리아지 라벨을 붙여라"), 이 표에서 대응하는 라벨 문자열을 쓴다.

로컬 마크다운 트래커에서는 이 문자열들이 각 이슈 파일 상단 근처의 `Status:` 줄에 적힌다.

오른쪽 열은 실제로 쓰는 어휘에 맞게 편집하라.
